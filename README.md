# Zoho CRM Skill - v3 Platform

This skill enables a Lua AI Agent to interact directly with Zoho CRM to manage leads, update statuses, and add notes. It is built using the **TypeScript v3 framework** for maximum reliability.

## Features
* **Lead Retrieval**: Fetch specific lead details using a Lead ID (e.g., LEAD-101).
* **Status Updates**: Update the stage of a lead in the CRM pipeline (e.g., 'Negotiation').
* **Note Management**: Add professional strategic notes directly to lead records.

## Technical Details
* **Framework**: Lua Platform v3 (TypeScript)
* **Validation**: Full Zod schema validation for all tool inputs.
* **Persona**: Configured as a Senior CRM Strategist.
* **Compliance**: Zero errors during `npm run typecheck`.
